## Angle
Apply Zero Trust principles specifically to AI/LLM environments — not generic ZT. This means:
- Never-trust/always-verify applied to agent-to-agent communication
- Microsegmentation for AI workloads and LLM API endpoints
- Identity-first security for Non-Human Identities (NHIs) — service accounts, API keys, agent tokens
- BeyondCorp-style access for LLM API consumers
- NIST SP 800-207 applied to AI infrastructure
- Continuous validation in agentic pipelines (not just at the perimeter)
- ZT + OWASP LLM Top 10 cross-reference
